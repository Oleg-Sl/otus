Запуска сервера производится командой: 
	python3 log_analyzer.py --config <путь к файлу конфигурации config.conf> 
если не указать параметр запустится с дефолтными параметрами.

Фаил конфигурации должен иметь  название "config.conf" и иметь следующую структуру:
REPORT_SIZE: <количество записей в отчете>
REPORT_DIR: <путь к директории хранения отчетов>
LOG_DIR: <путь к директории с файлами лога>
LOG_DIR_CURRENT_SCRIPT: <директория с файлом лога текущего скрипта>
PERCENT_ERRORS: <допустимый процент ошибок парсинга>
"TEMPLATE": <шаблон html-файла отчета>

Файлы анализируемого лога должны иметь имя (изменяться может только дата):
	"nginx-access-ui.log-20150629"
или
	"nginx-access-ui.log-20150629.gz"

Запуск тестирования функций log_analyzer.py:
	python test.py
